:::tip[pro tip]
Passbolt install MySQL as the default database management system, as detailed in the [installation guides](/hosting/install). After the package is installed, the setup wizard will assist in configuring the MySQL database. When using Docker, passbolt is pre-configured with a MariaDB container, meaning no additional database configuration is required in this setup.
:::
