Email server configuration (SMTP provider, credentials, sender address) is managed through the web UI. See the [Email Server configuration guide](/admin/emails/email-server/) in the Admin Guide for setup instructions with specific providers.

Email notification settings (which events trigger emails) are also configured in the web UI. See [Email Notifications](/admin/emails/email-notifications/).

If your emails are not being delivered after configuration, see the [troubleshooting guide](/hosting/email/troubleshooting/) for common SMTP issues and debugging steps.
