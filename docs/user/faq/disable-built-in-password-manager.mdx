Most web browsers and mobile devices include built-in password management that prompts you to save passwords for sites that you visit.

We will see in this help page how to disable this feature in web browsers and set Passbolt as default password manager on iOS / Android to avoid confusion and enhance security.

<Tabs>
  <TabItem value="chrome" label="Google Chrome" default>
    * Go to **chrome://settings/autofill** and select **Password Manager**
    * Turn off **Offer to save passwords** and **Auto Sign-in**.

    <Figure
      src="/img/help/2022/11/password-built-in-chrome.png"
      alt="Disable Google Chrome built-in password manager"
      caption="fig. Disable Google Chrome built-in password manager"
    />

  </TabItem>

  <TabItem value="firefox" label="Mozilla Firefox">
    * Go to **about:preferences#privacy**
    * Scroll down to **Logins and Passwords** menu
    * Uncheck **Ask to save logins and passwords for web sites**

    <Figure
      src="/img/help/2022/03/password-built-in-firefox.jpg"
      alt="Disable Mozilla Firefox built-in password manager"
      caption="fig. Disable Mozilla Firefox built-in password manager"
    />
  </TabItem>

  <TabItem value="edge" label="Microsoft Edge">
    * Go to **edge://settings/passwords**
    * Turn off **Offer to save passwords**

    <Figure
      src="/img/help/2022/03/password-built-in-edge.jpg"
      alt="Disable Microsoft Edge built-in password manager"
      caption="fig. Disable Microsoft Edge built-in password manager"
    />
  </TabItem>

  <TabItem value="brave" label="Brave">
    * Go to **brave://settings/passwords**
    * Turn off **Offer to save passwords** and **Auto Sign-in**.

    <Figure
      src="/img/help/2022/11/password-built-in-brave.png"
      alt="Disable Brave built-in password manager"
      caption="fig. Disable Brave built-in password manager"
    />
  </TabItem>

  <TabItem value="ios" label="iOS">
    ## Disable iCloud Keychain

    iCloud Keychain keeps informations like your Safari usernames and passwords, credit cards and Wi-Fi passwords up to date on any Apple device you approve.

    You can disable it if you want these data located only on Passbolt.

    * From settings, tap you name:

    <Figure
      src="/img/help/2022/03/password-ios-01.jpg"
      alt="iOS settings"
      caption="fig. iOS settings"
    />

    * Select iCloud:

    <Figure
      src="/img/help/2022/03/password-ios-02.png"
      alt="iCloud"
      caption="fig. iCloud"
    />

    * Select Keychain:

    <Figure
      src="/img/help/2022/03/password-ios-03.png"
      alt="Keychain"
      caption="fig. Keychain"
    />

    * Turn off iCloud Keychain

    <Figure
      src="/img/help/2022/03/password-ios-04.png"
      alt="Turn off iCloud Keychain"
      caption="fig. Turn off iCloud Keychain"
    />

    ## Verify AutoFill settings

    * Go to Settings > Passwords > AutoFill Passwords
    * Select Passbolt in **Allow filling from**

    <Figure
      src="/img/help/2022/03/password-autofill-iOS.jpg"
      alt="Configure autofill on iOS"
      caption="fig. Configure autofill on iOS"
    />
  </TabItem>
