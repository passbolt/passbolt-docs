## Prerequisites

Before setting up the passbolt mobile app on iOS, ensure you have:

- A passbolt account already configured in your browser
- Your passbolt server URL

:::warning
passbolt mobile apps require HTTPS to function. If you're using a self-signed certificate, you'll need to import it to your device first. See our [TLS (SSL) certificate import guide](/hosting/faq/how-to-import-ssl-certificate-on-mobile-application/) for detailed instructions.
:::

## Initial Setup

### Step 1: Launch the App

Open the passbolt app and tap "Connect to an existing account".

<Figure
  src="/img/help/2025/12/welcome_mobile.png"
  alt="Welcome screen of the passbolt mobile app"
  size={{ width: "320px" }}
  caption="fig. Welcome screen"
/>

### Step 2: Choose Setup Option

Select **"Connect to an existing account"**.

<Figure
  src="/img/help/2025/12/create_mobile.png"
  alt="Login options screen in the passbolt mobile app"
  size={{ width: "320px" }}
  caption="fig. Choose setup option"
/>

:::warning
It is currently not possible to create an account using the mobile app. You must first create an account using your desktop browser extension.
:::

### Step 3: Connect to Existing Account

1. **Sign in using the browser extension** on your desktop
2. **Go to the profile page** in your browser
3. **Go to the mobile section** in your profile

<Figure
  src="/img/help/2025/12/browser_mobile.png"
  alt="Mobile setup page in the browser showing QR codes"
  size={{ width: "600px" }}
  caption="fig. Mobile setup page in browser"
/>

### Step 4: Scan QR Code

1. In the passbolt mobile app, tap **"Scan QR code"**
2. Point your camera at the QR code displayed in your browser

<Figure
  src="/img/help/2025/12/camera_mobile.png"
  alt="Camera screen for scanning QR code in the passbolt mobile app"
  size={{ width: "320px" }}
  caption="fig. Scan QR code"
/>

### Step 5: Enter Passphrase

Enter your passphrase to authenticate and complete the setup.

<Figure
  src="/img/help/2025/12/login_mobile.png"
  alt="Enter passphrase screen in the passbolt mobile app"
  size={{ width: "320px" }}
  caption="fig. Enter passphrase"
/>

### Step 6: Enable Face ID

Enable Face ID (recommended) so you can unlock the app with your face.

<Figure
  src="/img/help/2025/12/faceid_mobile.png"

Configure autofill settings. See our [guide on disabling built-in password managers](/user/faq/disable-built-in-password-manager/).

## Disable Built-in Password Manager

Disable iOS's built-in password manager to prevent conflicts. See our [guide on disabling built-in password managers](/user/faq/disable-built-in-password-manager/).

## Troubleshooting

### Certificate Issues

- Ensure your server uses HTTPS
- For self-signed certificates, import the certificate to your device
- Check our [TLS (SSL) certificate troubleshooting guide](/hosting/faq/how-to-import-ssl-certificate-on-mobile-application/)

### Autofill Not Working

If autofill is not working, check that:
- The passbolt autofill service is enabled in your device settings
- You have granted necessary permissions to the passbolt app
- Your device's built-in password manager is disabled
